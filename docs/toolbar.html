Toolbar

Summary

Toolbar is superceded by a combination of Buttons and Toolbar2. Toolbar implementations should be replaced with Toolbar2. This documentation is provided for the maintenance period.

Code

Toolbar's API is HTML (Soy is available to help implement the HTML API).

Toolbar layout is based on two concepts: toolbar groups and toolbar splits.

  • Toolbar groups are used to enclose sets of related buttons, causing them to visually combine - this makes them a toolbar instead of several separate buttons. If you want a single button, make a toolbar group of just one button.
  • Toolbar splits are used to split up toolbars into major areas - eg. left and right button sets, rows, etc.

Inside toolbar groups, you'll find each button is a trigger element. That is, interacting with that element is what triggers events.

You can set a disabled style on a Toolbar trigger by adding the class disabled to the group, item or trigger - if in doubt, apply it to the toolbar-item.


<ul class="toolbar-group">
    <li class="toolbar-item disabled">
        <a class="toolbar-trigger" href="#">Off</a>
    </li>
    <li class="toolbar-item">
        <a class="toolbar-trigger" href="#">On</a>
    </li>
</ul>

Note this does not disable any Javascript you have bound to the trigger, it simply sets a disabled style. Disabled toolbar styles can be combined with the ability to disable Dropdowns.

The toolbar component only styles the buttons, for the sake of flexibility toolbar containers are styled at the product level.

Toolbar v1 was often combined with the also-deprecated Dropdown v1:


<li class="toolbar-item toolbar-dropdown">
    <div class="aui-dd-parent">
        <a href="#" class="toolbar-trigger aui-dd-trigger">LI Dropdown</a>
        <ul class="aui-dropdown">
            <li class="dropdown-item"><a href="#" class="item-link">Dropdown item</a></li>
            <li class="dropdown-item"><a href="#" class="item-link">Dropdown item</a></li>
            <li class="dropdown-item"><a href="#" class="item-link">Dropdown item</a></li>
            <li class="dropdown-item"><a href="#" class="item-link">Dropdown item</a></li>
        </ul>
    </div>
</li>
