Loading the flat pack

What's in this flat pack?

This flat pack contains both core/stable components from AUI, plus some components with experimental APIs (in separate resources). The flat pack is great for rapid prototyping - the code here is the code that's shipping in AUI (and hence in Atlassian products).

Download the AUI flat pack

How to load this flat pack

If you want everything including AUI's dependencies, Experimental features and compiled-JS Soy templates, add this to the HEAD of your templates or page:

<link rel="stylesheet" href="aui/css/aui-all.css" media="all">
<!--[if lt IE 9]><link rel="stylesheet" href="aui/css/aui-ie.css" media="all"><![endif]-->
<!--[if IE 9]><link rel="stylesheet" href="aui/css/aui-ie9.css" media="all"><![endif]-->
<script src="aui/js/aui-all.js"></script>
<!--[if lt IE 9]><script src="aui/js/aui-ie.js"></script><![endif]-->

...remembering to update the path to point to the location of your copy. This is the recommended/default option.

For more granular control (eg. to skip Experimental or drop IE8 support), you can use the appropriate combination of these lines (note the dependencies are required):

<link rel="stylesheet" href="aui/css/aui.css" media="all">
<link rel="stylesheet" href="aui/css/aui-experimental.css" media="all">
<!--[if lt IE 9]><link rel="stylesheet" href="aui/css/aui-ie.css" media="all"><![endif]-->
<!--[if IE 9]><link rel="stylesheet" href="aui/css/aui-ie9.css" media="all"><![endif]-->
<script src="aui/js/aui-dependencies.js"></script>
<script src="aui/js/aui.js"></script>
<script src="aui/js/aui-experimental.js"></script>
<script src="aui/js/aui-soy.js"></script>
<!--[if lt IE 9]><script src="aui/js/aui-ie.js"></script><![endif]-->
