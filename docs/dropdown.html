Summary

JavaScript

  • type - A string parameter that determines the type of dropdown being created
  • options - A number of different options may be passed in as an associative array

See Options for types and options.


AJS.$(document).ready(function() {
    AJS.$("#dropDown-standard").dropDown("Standard");
});

Options

OptionDetailsExamples
itemThe tagName of the individual menu items as a string. 
Accepts any valid HTML Tag (Normally li or a).
Default: "li:has(a)"

AJS.$("#dropDown1").dropDown("Standard", {item: "a"});
alignmentDetermines if the dropdown is aligned to the left of the trigger or the right. 
Possible values: "left"/"right". 
Default: "right"

AJS.$("#dropDown1").dropDown("Standard", 
    { alignment:"right" }
);
escapeHandlerA function that runs when the "ESC" key is hit while the dropDown is open/visible.
Accepts any Javascript function. 
Default: function () { this.hide("escape"); return false; }

AJS.$("#dropDown1").dropDown("Standard", { 
        escapeHandler: function() { 
        alert("Hello World"); 
        } 
    }
);
hideHandlera function that runs after the dropdown is hidden. 
Accepts any javascript function. 
Default: function() {}

AJS.$("#dropDown1").dropDown("Standard", 
    { 
        hideHandler: function() { 
        alert("Hello World"); 
        } 
    }
);
activeClassSpecifies a class to apply to the active element item of the dropdown. 
Accepts a string value. 
Default: "active"

AJS.$("#dropDown1").dropDown("Standard", 
    {
        activeClass: "IamActive" 
    }
);
useLiveEventsAUI supports jQuery live events. If you choose this option, AUI will bind all events on the page to the HTML body element instead of to each individual element. This means that your events can be bound to all current elements and to elements that do not yet exist. You no longer need to rebind everything on an Ajax load. This is essential on a page which has, for example, a number of user avatars that react on hover. Binding can cause a performance problem when there is a large number of such elements.See AUI 3.1 Release Notes.
useDiabledEnables the "disabled" class. Setting the value to true will disable any dropdown which has a class of disabled set on the aui-dd-parent. The dropdown can be re-enabled by removing the disabled class. 
Options: true/false. 
Default: false

AJS.$("#dropDown").dropDown("Standard", 
    {
        useDisabled: true
    }
);
